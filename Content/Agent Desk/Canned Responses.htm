Canned Responses

Canned responses allow you to prepare short scripts that agents can use when dealing with clients. For example, you may want to prepare different classes of script that agents can use to create leads or cross-sell services.

Canned Responses

You can define the scope as yourself (agent) or you can make the script available to all registered users in the account, or you can even make it available to particular teams.

When you use our analytical tools (Agent Monitor, Statistics) you may be able to draw correlations between scripts and successful sales. These short scripts can be written by any agent in the account and therefore, over time, you may be able to accumulate a set of ‘magic’ scripts that your staff have written and distribute them to the appropriate places.

 Preparing Canned Responses

Canned responses are shortcuts for agents. Over time Agents become familiar with the types of questions they are asked and the kinds of responses their company wants to offer, in light of their sales and service strategies. With canned responses agents can use pre-packed responses that they themselves have fashioned. When the time comes and a visitor has asked a question or made a remark that makes one of the canned responses appropriate, the agent only has to use a single key stroke to inject their canned response into a chat.

  1. From the main menu select Settings > Canned responses.
  2. Select the + New canned response button. A modal page slides in from the right.

Once you have prepared a script you need a way to insert it into a chat session. You do this by first defining a special shortcut key (from the keyboard) that will automatically generate the prepared script when you type. See Using the Canned Responses during a Session for instructions on using canned responses during a session.

Note: Try to use uncommon keys and key combinations. Here we have used the caret (^) key. You can use the same keyboard shortcut for more than one script. (When you want to enter a script during a session you will be offered all scripts available according to the defined scope.)

  1. Enter a (keyboard) key in the Shortcut field.
  2. Enter a descriptive title in the Title field.
  3. Write the script in the Canned text field.
  4. Define your Scope using the radiobuttons then click Save. The response is added to the main Canned responses page.

Note that you can edit the script at any time using the Edit () icon.

 Using the Canned Responses during a Session

When, as an agent involved in a session, you want to use a canned response within a chat, enter the ‘/’ (forward slash with no quotation marks) character into the chat box.

All of the canned responses available to you will appear in a popup.

Note: As in the picture above, you can use a single special character to hold as many responses as you wish.

Select the response you want to insert into the chat then click the Send button.
