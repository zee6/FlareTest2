Assigning Users to Teams

Note: You must have Admin user rights in order to assign users/agents to teams.

  1. Select Settings > Users. The Users page displays.

Choose the first user you want to assign to a team. In this example we will choose Registered User Ian Allsom.

  1. Select the pencil icon () to the right of the user’s name. The Edit user modal page slides in from the right of the screen.

  1. From the Team drop-down list select the team to which the user is to be assigned.

  1. Click the Save button. Now if you check the Users page you will see that the user has been added to the team.

Note: Members of a team do not show on the Teams page. To see which team a user is a member of go to the Users menu. You can group or filter users there to organize your teams.

  1. Repeat these steps until you have added all of your users to the appropriate teams.

Note: Registered users cannot see each other on the Users page. In fact, there is no ‘User’s page’ as far as a Registered user is concerned. However, Registered users can see other users when they want to forward a visitor in the Queue, or an already-started session, to another agent or team.
