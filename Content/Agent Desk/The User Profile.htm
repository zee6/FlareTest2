The User Profile

  1. All users can access their own profiles by selecting Settings > User profile from the left menu. The User Profile page displays.

In the User Profile page users can see and change their (own) details. This is useful if you have created a temporary password for the user and now they must change that password, or if an email address or user name should change.

  1. If a user wants to change their password then they can get to this page by selecting Settings > User Profile then selecting the Define password radiobutton. You are then required to enter the password twice.
  2. Click Save and it is done.
