Quick Start Guide for New Accounts (Deutsch)

Ich habe ein neues unblu Konto

Wenn Sie auf dieser Seite landen, haben Sie wahrscheinlich gerade ein neues unblu Konto eingerichtet. Wir bieten Ihnen eine ausführliche unblu Dokumentation auf unserer Website (in Englischer Sprache). Wir verstehen aber auch, dass die vielen Optionen die unser System zulässt zunächst verwirren können. Um Ihr neues Konto einzurichten, sollten Sie zunächst folgendes tun:

Melden Sie sich mit Ihren Nutzerdaten in Ihrem unblu Konto an. Nach dem Login sehen Sie unseren Agent Desk. Mit diesem können Sie Co-Browsing Sitzungen initiieren, Chat Anfragen annehmen und das System konfigurieren. Alle Agentenaktivität wird von diesem Arbeitsplatz aus gesteuert. Wenn Sie Administratorenrechte haben, nehmen Sie auch alle Kontoeinstellungen in diesem Interface. Ihr kontospezifisches Java Script Snippet können Sie ebenfalls hier finden.

Das unblu Snippet auf Ihrer Website

Um die unblu Funktionalität auf Ihrer Website zu aktivieren müssen Sie eine Java Script Snippet in den Header Ihrer Website einfügen. Ihr Kontospezifisches Snippet können Sie im Agent Desk unter Kontoeinstellungen finden: https://start.unblu.com/unblu/desk#/settings/account/apikeys. Klicken Sie auf den Stift im API Key Eintrag um zum Snippet zu gelangen. Um diese Seite aufzurufen benötigen sie Adminstrative Rechte für das System. Sobald Sie eine Webseite mit dem unblu Snippet instrumentiert haben, vergessen Sie bitte nicht die Domäne in den Kontoeinstellungen zu registrieren.

Ihre Lizenz verwalten:

Wenn Sie ein unblu starter Konto haben, können Sie unblu nur auf http Sites verwenden. Sollten Sie auch https Sites verwenden wollen, können Sie via das Gear Menü die Lizenzverwaltung ansteuern und eine höhere Lizenzart auswählen.
